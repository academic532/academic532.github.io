Python Kit

Python Kit

This section provides a guide for configuring Python environment on your VPS.

The name of the game

Resolving dependences of Python packages could be very painful, Anaconda is a better practice of configuring scientific Python enviornment on your machine, since it integrates popular packages as bundles and distributes package versions using a powerful package manager - conda.

The rest of this section will guide you to install your own Anaconda distribution, which would not confict with other user's Python environment.
