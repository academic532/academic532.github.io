pacman包管理器

pacman 包管理器

pacman 是 Archlinux 的默认包管理器，常用的软件包均可通过 pacman 指令安装、升级和卸载。

安装

$ sudo pacman -S [package_name]

其中 [package_name] 为需要安装的软件包的名字。可以通过 pacman -Q 指令查询软件包，或通过 Packages-ArchLinux 页面进行软件包的检索。

卸载

$ sudo pacman -R [package_name]

升级

$ sudo pacman -Syyu
